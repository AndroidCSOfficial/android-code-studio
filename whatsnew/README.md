# What's New

### Android NDK Support
Added support for the Android Native Development Kit (NDK), enabling the compilation of native code. While the development environment runs on `aarch64` Android devices, it can target all standard Android architectures.

### Auto-Save Feature
A new auto-save functionality has been implemented to safeguard your work. Once enabled, all edits are saved automatically, preventing data loss in the event of an unexpected application closure.

**To enable:**
Navigate to **Preferences > Editor > Auto Save** and toggle the switch to the on position.
